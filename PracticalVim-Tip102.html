Simple Case: Execute catgs Manually

:!ctags -R
  • create a mapping: 
    :nnoremap <f5> :!ctags -R<CR>

Automatically Execute ctags Each Time a File is Saved

:autocmd BufWritePost * call system("ctags -R")
