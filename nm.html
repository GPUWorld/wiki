Common Usage

# output: symbol value, symbol class, symbol name
$ nm a.o
# sort in a reversed order
$ nm -r a.o
$ nm --reverse-sort a.o

sort by size 

$ nm --size-sort a.o
$ nm --size-sort -r a.o
