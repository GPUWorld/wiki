strip

% toc

Common Usage

delete all symbols 

$ strip a.out
# is equivalent to 
# objdump --strip-*

delete symbols for debugging 

$ strip -d a.out

delete symbols of some section 

# delete symbols of .text section
# make the program unable to run
$ strip -R .text a.out
