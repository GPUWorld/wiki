Tip 17: Insert Unusual Characters by Character Code

Return to the top: <<Practical Vim>>

Vim can insert any character by its numeric code, and expects the numeric code to consist of three digits: <C-v>{code} e.g. Insert "A" in insertion mode: <C-V>065
