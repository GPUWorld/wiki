static library

Create archive file (*.a): 

$ cc -c -o foo.o foo.c
$ cc -c -o bar.o bar.c
$ ar ruv libfoo.a foo.o bar.o

Inside the archive file: 

$ ar tv libfoo.a

Link: 

$ cc -o baz baz.o -lfoo

shared library

Create shared library: 

$ cc -fPIC -c -o foo.o foo.c
$ cc -fPIC -c -o bar.o bar.c
$ cc -shared -W1,-soname,libfoo.so.0 -o libfoo.so foo.o bar.o

Link: 

$ cc -o baz baz.o -lfoo
